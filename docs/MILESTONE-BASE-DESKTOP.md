# Milestone: Base Desktop

This milestone proves that Kaizen Linux can turn a fresh Fedora Workstation install into a working Fedora Hyprland desktop.

## Confirmed working

- Fedora 44 base
- RPM Fusion setup
- Hyprland install path
- xdg-desktop-portal-hyprland
- Kitty terminal
- Rofi launcher
- Waybar panel
- Fastfetch branding
- Default wallpaper
- Papirus icon theme package profile
- JetBrains Mono / Noto font package profile
- VM-safe fallback keybinds

## Known notes

- hyprpaper is currently optional because package availability can vary.
- swaybg is used for the default wallpaper.
- Quickshell / AetherDeck is not part of the base milestone yet.
- Gaming packages are separate and should be tested after the desktop base is stable.

## Next milestone

Add gaming and productivity profiles, then begin ISO/Kickstart work.
