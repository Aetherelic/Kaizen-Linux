# Kaizen Linux v0.2.0-alpha-test3 Test Result

## Result

v0.2.0-alpha-test3 is the first successful full-rice-integration ISO test.

## Confirmed Working

- ISO boots into live Hyprland session.
- Calamares install completes.
- Installed system boots without manual BLS repair.
- Installed system identifies as Kaizen Linux.
- Installed user receives Kaizen Hyprland configs.
- Super + T opens Kaizen Tools.
- Kaizen Tools desktop launcher appears and works.
- Super + Shift + A opens the Kaizen wallpaper picker.
- Wallpaper picker successfully changes wallpaper.
- swaybg is used as the Fedora-safe wallpaper backend.
- Quickshell does not auto-start.
- awww-daemon does not auto-start.

## Known Remaining Polish

- GRUB/BLS boot menu still displays Fedora Linux instead of Kaizen Linux.
- Fedora media check may warn on boot for custom ISO.
- Quickshell/AetherDeck are intentionally disabled until packaged properly.
- Full visual wallpaper grid from the personal rice is not yet ported.
