# Aetherelix Linux First Boot

Welcome to the early Aetherelix Linux Fedora Hyprland test image.

## Default keybinds

- Win + Enter: open terminal
- Win + Q: close active window
- Win + M: exit Hyprland
- Win + R: reload Hyprland
- Alt + Space: open app launcher
- Win + H/J/K/L: move focus
- Win + Shift + H/J/K/L: move window
- Win + F: fullscreen
- Win + V: toggle floating
- Win + 1-5: switch workspaces
- Win + Shift + 1-5: move window to workspace

## VM fallback keybinds

Some host systems capture the Win key inside virtual machines.

Fallback binds:

- Ctrl + Alt + T: open terminal
- Ctrl + Alt + Q: close active window
- Ctrl + Alt + M: exit Hyprland
- Ctrl + Alt + R: reload Hyprland
