# Kaizen Linux v0.1.0-alpha

## Status

First successful installable Kaizen Linux alpha ISO.

## Confirmed working

- ISO boots in UEFI mode
- Live environment boots into Hyprland
- No TTY login blocker
- No live password prompt blocker
- Kaizen wallpaper appears
- Waybar appears
- Rofi launcher works
- Kitty terminal works
- Calamares installer launches
- Calamares branding appears
- Installation completes successfully
- Installed system reboots into Hyprland
- Installed system identifies as Kaizen Linux
- Default Kaizen wallpaper is included

## Known limitations

- Secure Boot is not supported yet
- Test VM must use UEFI without Secure Boot
- Wallpaper selector is not fully implemented yet
- Quickshell/AetherDeck is not fully packaged yet
- Some Fedora branding may still remain in lower-level tooling
- Gaming/productivity profiles need more polish

## Next goals

- Improve Calamares wording and branding
- Add proper Kaizen first-boot/welcome screen
- Package Quickshell/AetherDeck properly
- Add wallpaper/theme selector
- Add gaming and productivity profile selection
- Clean remaining Fedora labels
