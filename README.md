# Kaizen Linux Fedora Remix

A Fedora-based Hyprland desktop project focused on:

- Hyprland
- Quickshell / AetherDeck
- Gaming performance
- Productivity tools
- Clean visual design
- Fedora stability

## Current status

Early development.

The first goal is to create a reliable post-install script that turns a fresh Fedora install into the Kaizen Linux desktop.

## Roadmap

1. Post-install script
2. Fedora VM testing
3. Hyprland desktop setup
4. Gaming package profile
5. Productivity package profile
6. Optional gaming kernel support
7. Fedora Remix ISO
